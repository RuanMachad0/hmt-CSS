Exercicío de CSS

Organizando

Utilixando CSS para melhorar a visibilidade do site e criar um ambiente mais interativo

<main>

um codigo HTML que usamos para fazer uma marcação, e com a utilização de CSS criamos essa área destacada que ira hospedar o nosso conteudo.

Gradiente

Gradiente é um um degradê de cores como se pode ver ao fundo desse exercicío.

Codigos

Veja como o CSS foi colocado no <main> 

 main {   
   background-color: rgb(255, 255, 255);
   border-radius: 10px;
   box-shadow: 5px 5px 15px #89babb75;
   width: 500px;
   padding: 10px;
   margin: auto;
 }

Veja como foi feito o grafismo

  background-image: linear-gradient(to bottom, cor);
  background-attachment: fixed;
