pseudo-classes

visited

Visited deixa os links que já foram visitados com algum detalhe diferente.

Exemplo

a:visited {
   color: green;
}

active

Active faz que qquando tal codogo cor ativado(clicado) seja mudado algo.

Exemplo

a:active {
   color: red;
}

before/after

Retrado como antes e depois, utilizado para colocar algo antes ou depois de algum codigo para complementação.

Exemplo

a::before {
   content:'-';
}

a::after {
   content:'[link]';
   color: #7272706c;
}

hover

Já sabemos como ele funciona porem iremos utiliza-lo também para persolalizar link.

Exemplo

a:hover {
   color: aqua;
}
