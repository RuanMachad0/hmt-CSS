Exercicío com Fontes

Explicação

font-family

Para utilizar fontes diferentes da padrão é necessario utilizar um codigo CSS, sendo ele "font-family" que permite modificar apenas fontes(letras).

font-size

quando se trata de tamanho da fonte utilizamos "font-size", abaixo você poderá ver a tabela de tamanhos que exitem para que se possa usar:

    Medidas Absolutas
  • cm = centimetro
  • mm = minimetro
  • in = polegada
  • px = pixel
  • pt = ponto
  • pc = paica

    • Medidas Relativas
  • em = tamanho atual
  • ex = altura x
  • rem = altura configurada no body
  • vw = largura da viewport
  • vh= altura da viewport
  • % = porcentagem

font-weight

font-weight se trata de peso(Grossura) das letras, abaixo poderá ver uma lista das opções de pesos:

  • lighter = fina
  • normal = normal
  • bold = negrito
  • bolder = maior que bold
  • 100-900 = opção ajustavel

Dica importante para abreviar CSS

Inves de colocar isto:

 font-size: 1em;
 font-style: italic;
 font-weight: 900;
 font-family: sans-serif;

Utilize isso:

 /*font-size font-weight font-size font-family*/
 font: italic 900 1em sans-serif;
