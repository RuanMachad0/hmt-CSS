Exercicío 02 sobre centralização vertical

Centralização vertical

Teste

O que é

Seria centralizar uma imagem dentro de outra, para que isso aconteça a primeira imagem deve estar contida em um bloco com posição relativa, a outra imagem deve também estar dentro do mesmo bloco porém contida em mais um bloco com uma posição absoluta.

E para que isso ocorra também utilizamos um novo código CSS sendo ele transform.

Exemplo

section#secao {

   position: relative;

   height: 95vh;

   padding: 10px;

   background-image: url('../../Exer-01/imagens/target001.png');

   background-size: 100% 100%;

   background-color: rgb(255, 123, 0);

}


article#artigo {

   background-image: url('../../Exer-01/imagens/target001.png');

   background-size: 100% 100%;

   position: absolute;

   height: 200px;

   min-width: 400px;

   background-color: rgba(0, 17, 255, 0.918);

   left: 50%;

   top: 50%;

   transform: translate(-50%, -50%);

}
