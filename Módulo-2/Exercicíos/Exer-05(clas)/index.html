Exercicío de classificação

Identificação

Para identificar algum codigo html é utilizado id="" assim podendo ser alterado em particular, e para se referir a id em CSS é utlizado "#".

Exemplo

<p id="teste"></p>
  @charset "UTF-8";

p#teste {
    text-indent: justify;
}

Classificação

Para classificar algum codigo utilizamos class="", e para modifica-lo com CSS é utilizado ".".

Exemplo

<p class="teste1"></p>
<strong class="teste2"></strong> 
<ins class="teste2 teste1"></ins>

@charset "UTF-8";

.teste1 {
    text-indent: justify;
}
.teste2 {
    color: #7fe7e9;
}

Diferença entre "ID" e "CLASS"

A diferença entre esses dois que se até parecem iguais mas não são, "ID" é unico não se pode utilizado para varios codigos apenas um, mesmo que eles funcione para varios, são regras.

Já o "CLASS" tem liberdade total, pode ser utilizado em varios codigos sem limite, podendo tambem utilizar mais do que uma classificação em um codigo, assim sendo mais pratico e simples de ser utilizado dentro de seus programas.
