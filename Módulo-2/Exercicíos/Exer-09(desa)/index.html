Correção do desafio 10

Erros

Header

Header é apenas para o titluo principal do site.

article

Article é todo o conteudo do site.

Nav

Nav é para criar um block de navegação, para opções de menu por exemplo

Global

utilizando * para dar um estilo global, exemplo no site foi usado margin 0px e padding 0px.

Codigos importantes

root

root é uma tag CSS que é utilizado para criar variaveis do proprio CSS.

Exemplo

        :root {
          --cor01: white;
        }

width

alguns tipos de width uteis

  • 100% = um tamanho que ocupa 100% do espaço, se adapta tela.
  • min-width = tamanho mínimo que pode chegar.
  • max-width = tamanho máximo que pode chegar.

line-height

serve para criar um espaço vazio entre as linhas.

Tags de lista

  • list-style-type = tipo do caractere que fica a esquerda de cada linha da lista.
  • list-style-position = posição das linhas da lista.
  • columns = quantas colunas a lista irá ter.

Responsividade

renponsivel quer dizer um site que respondem bem em qualquer tamanho de telas.
