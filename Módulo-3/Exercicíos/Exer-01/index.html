Tipos de background

Até o momento já vimos "background-color:;" que inclui uma cor ao fundo do site, "background-image: linear-gradient();" que coloca um gradient ao fundo do site e por fim "backgrond-image:;" que tranforma uma imagem em um fundo de tela.

background-repeat

Este codigo de CSS da uma propriedade ao background-image, sendo se a imagem for menor que o fundo que a imagem foi aplicada ela será replicada até encher a o fundo sempre iniciando no canto superior esquerdo.

  • Tags
  • repeat = repita
  • no-repeat = não repita
  • repeat-x = repita em uma linha
  • repeat-y = repita em uma coluna

Exemplo

background-repeat: no-repeat;

background-position

Já este codigo CSS permite ajustar o inicio da imagem, para as imagens que são maiores do que o fundo que foram aplicadas.

    Tags
  • left top = inicio padrão
  • left center = esquerda ao centro
  • center center = centralizado
  • right bottom = canto inferior direito

Exemplo

Background-position: center center;

background-size

Codigo CSS que define tamanho á iamgem que fica ao fundo no background.

    Tags
  • %,px... = tamanho padrão
  • contain = tamanho da view port porém acaba sendo distorcida
  • cover = tamanho da view port sem distorções

Exemplo

Background-size: contain;

background-attachment

Este codigo CSS define o comportamento do backgrond quando se "rola" a tela.

    Tags
  • scroll = se movimenta junto com o conteudo
  • fixed = se mantem fixo

Exemplo

Background-attachment: fixed;

short-hand background

Resumindo os codigos acima, nessa sequencia:

color -> image -> position -> repeat -> size -> attachment

Exemplo

Background: black url('../imagens/wallpaper002.jpg') center center
no-repeat contain fixed;
